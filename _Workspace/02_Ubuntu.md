# Debian package
```
sudo apt install ./<file>.deb
```

# Export to bashrc
```
echo 'export PATH="/usr/local/opt/qt/bin:$PATH"' >> ~/.bashrc
```

# Brew [https://brew.sh/](https://brew.sh/)
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install wget

```
